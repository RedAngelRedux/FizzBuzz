Instructions

Enter two numbers, select a range, and click the button.

Expected Result

Our busy little bee will display all the numbers in your range, replacing the number with "fizz" if it is divisible by the first input, "buzz" if it is divisible by the second input, and "fizzbuzz" if divisible by both.

Logic Variations

Four different solutoins are implemented for FizzBuzz and you can choose which solution to run by selecting A - D before executing. For details on each solution, refer to the codee page.
